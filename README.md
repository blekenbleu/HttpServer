## [HttpServer](http://127.0.0.1:7734/) using [TCPListener](https://learn.microsoft.com/en-us/dotnet/api/system.net.sockets.tcplistener?view=netframework-4.8)
 &emsp; *from [simple-HttpServer](https://github.com/Peteri-git/simple-HttpServer)*  
- .NET Framework 4.8 `.exe` using `TcpClient()`, `GetStream()`, `StreamReader()` and `StreamWriter()`
	- a single `static class` with `Thread(Process)`
		- displays a Menu for [default GET](http://127.0.0.1:7734/) or file from R:\TEMP\
		- displays a file from Menu POST
- [**HTTP - Responses**](https://www.tutorialspoint.com/http/http_responses.htm)
- [HTTP Response - w3.org](https://www.w3.org/Protocols/HTTP/Response.html)
